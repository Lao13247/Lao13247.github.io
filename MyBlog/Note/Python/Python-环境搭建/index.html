# Python 开发环境搭建（一）

Anaconda 和 Pycharm 两个可以作为 Python 集成开发环境，Anaconda 可以很方便地管理不同版本的 python 语言环境和依赖包的导入等，Pycharm 则是 Jetbrains 公司发行的 python 工程集成开发软件，功能强大，能够胜任各种需求的开发任务。
