# 前言

Hexo 框架是基于 NodeJS 构建的博客框架，可以快速高效地解析 Markdown 语言编写的文本文件，生成博客网站的各种静态网页内容，还能通过主题和功能插件等实现定制化的样式和功能。

# 环境准备

搭建 Hexo 博客框架需要先安装 NodeJS 和 git 工具。

NodeJS 是 Hexo 框架解析文件的必需环境，安装后可以使用 npm 命令安装所需的功能插件；git 工具则是用来下载主题、线上部署等。

Node.js 官网下载：http://nodejs.cn/download/

git 官网下载：https://git-scm.com/download
